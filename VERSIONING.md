# Versioning Policy

This project follows [Semantic Versioning 2.0.0](https://semver.org/).

## Summary
* **MAJOR (x.0.0):** Breaking changes to the `contracts/` or the Conformance API.
* **MINOR (0.x.0):** New capabilities, new layers in `docs/`, or new non-breaking features.
* **PATCH (0.0.x):** Typos in docs, bug fixes in `tools/`, or metadata updates.

## Current Status
Current Version: **0.1.0-alpha** (Pre-release / Initial Specification Phase)
